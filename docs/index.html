sparkta

Interactive, self-contained HTML charts from Stata.
One command. Zero dependencies. No Python. No R. No server.

⚡

One command

Runs inside your existing do-file. No Python environment, no R packages, no context switch.

📁

Single file output

A self-contained .html file. Email it, archive it, put it on a USB drive. No dependencies for the viewer.

🔒

Data never leaves Stata

All processing runs locally via the Java Plugin Interface. No uploads, no telemetry, no external API.

📶

Fully offline capable

The offline flag bundles all JavaScript into the HTML. Zero network requests when opened.

⛰

500k obs in ~1.2 s

O(N) group scan, bulk SFI reads, lazy sparkline rendering. Fast even on large research datasets.

📊

CI charts matching Stata

t-distribution intervals and summary statistics computed inside Stata. Always match your results window.

Live chart gallery

All charts below are real sparkta output from Stata's built-in auto and nlsw88 datasets. Hover, click the legend, and use the dropdowns -- everything is interactive.

CI Bar Chart

Mean car price with 95% t-distribution confidence intervals by repair record. Statistics match Stata ci means exactly.

Violin Plot

Animated kernel density estimate with IQR box and Tukey whiskers. Click play to re-animate. Unique to sparkta among Stata packages.

Dual Interactive Filters

Two independent dropdown filters -- Foreign/Domestic and headroom -- filtering the same chart live. No server. No JavaScript framework.

Reference Annotations

Horizontal reference line labelled with ylinelabel() and a shaded reference band with yband().

Scatter with Over Groups

Price vs MPG, coloured by domestic vs foreign origin. Hover any point for exact coordinates.

100% Stacked Bar

Wage composition by race group stacked to 100%. Immediately reveals share rather than absolute values.

Panel Layout with by()

Separate panels for domestic and foreign cars using by(foreign). Each panel is independently interactive.

Dark Theme

Dark background mode using theme(dark). All palettes work on both light and dark backgrounds.

For sensitive and restricted data

All chart generation runs inside Stata on your local machine. The output HTML has no data-fetching code, no callbacks, and makes zero external network requests. The offline option embeds all JavaScript directly -- suitable for IRB-restricted data, clinical records, financial data, and institutional networks that block external CDN requests. 

* Install
ssc install sparkta

* Basic bar chart grouped by repair record
sysuse auto, clear
sparkta price, over(rep78)

* CI bar chart -- t-distribution intervals matching ci means exactly
sparkta price, type(cibar) over(rep78) cilevel(95)

* Violin plot with animated KDE density
sparkta price, type(violin) over(rep78)

* Interactive filter dropdown -- no server required
sparkta price, over(rep78) filter(foreign)

* Dual filter dropdowns
sparkta price, over(rep78) filter(foreign) filter2(headroom)

* Fully air-gapped output -- all JS embedded, zero network requests
sparkta price, type(cibar) over(rep78) ///
    offline export("~/secure/chart.html")
